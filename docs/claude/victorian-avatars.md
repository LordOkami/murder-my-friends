# Avatares Victorianos - Murder My Friends

## Resumen

32 retratos victorianos (16 masculinos + 16 femeninos) para usar como fotos de perfil predefinidas.

## Especificaciones de imagen

- **Formato:** WebP (preferido) o PNG
- **Tamano:** 512x512 px cuadrado
- **Peso objetivo:** < 80 KB cada una (WebP)
- **Carpeta destino:** `img/avatars/`
- **Nomenclatura:** `m01.webp` a `m16.webp` (masculinos), `f01.webp` a `f16.webp` (femeninos)

## Prompt base (prefijo comun para todos)

```
Victorian oil portrait, bust shot, dark moody background with subtle warm tones,
painterly brushstrokes, dramatic chiaroscuro lighting, 19th century attire,
richly detailed fabric textures, square format, no frame, no text
```

> Pega este prefijo antes de cada prompt individual.

2. Pedirle a Claude que implemente el selector de avatar con galeria en modal
3. El selector sera una alternativa mas junto a "subir foto" y "tomar foto webcam"
